# Choriot Logo Design Brief

## Brand Name
**Choriot** - inspired by a Roman Chariot that carries chores to completion

## Logo Concept
A playful, cartoon-style chariot loaded with cleaning supplies

## Design Prompts for AI Image Generators

### Main Prompt (Recommended)
```
A cute cartoon logo of a Roman chariot filled with cleaning supplies. The chariot has large wooden wheels with spokes and is carrying a mop, broom, bucket, spray bottle, and sponge. Bright, cheerful colors with a playful style. Clean lines, simple shapes, suitable for an app icon. Transparent background.
```

### Alternative Prompts

**Option 1: More Detail**
```
Minimalist cartoon logo featuring a front corner view of a Roman-style chariot with two large wooden wheels. The chariot basket is filled with household cleaning items: a blue bucket with soap bubbles, a yellow mop, a brown broom, a green spray bottle, and a colorful sponge. Friendly, approachable design with smooth curves. Bright but not oversaturated colors. Modern flat illustration style perfect for a mobile app icon. Transparent background.
```

**Option 2: Dynamic/Action**
```
Playful cartoon chariot logo in motion, wheels spinning, carrying cleaning supplies (mop, broom, bucket, spray bottle) with soap bubbles trailing behind. Cheerful color palette with blues, yellows, and greens. Simple, bold shapes that work at small sizes. Modern vector illustration style. Clean design suitable for app branding.
```

**Option 3: Icon-focused**
```
App icon design: simplified cartoon chariot with cleaning tools, circular composition, bold outlines, limited color palette (3-4 colors), recognizable at small sizes, professional yet fun, modern illustration style
```

## Design Specifications

### Style Guidelines
- **Aesthetic**: Cartoon, playful, friendly, approachable
- **Complexity**: Simple enough to be recognizable at small sizes (app icon)
- **Line Style**: Clean, smooth, rounded edges
- **Color Palette**: Bright and cheerful but professional
  - Browns/tans for chariot wood
  - Blue for bucket/water elements
  - Yellow/green for cleaning tools
  - Consider soap bubbles for whimsy

### Technical Requirements
- Works well at multiple sizes (16px to 1024px)
- Clear silhouette for recognition
- Transparent or white background
- High contrast for visibility
- Suitable for both light and dark UI themes

### Elements to Include
1. **Chariot**: Roman/ancient style with visible wheels
2. **Cleaning Supplies** (choose 3-5):
   - Mop
   - Broom
   - Bucket (possibly with water/bubbles)
   - Spray bottle
   - Sponge
   - Duster
   - Soap bubbles (decorative)

### Mood & Personality
- Efficient and capable
- Friendly and approachable
- Slightly whimsical/fun
- Trustworthy and professional

## Usage Instructions

### For DALL-E 3 / ChatGPT
1. Copy the main prompt or an alternative
2. Add: "Generate in PNG format at 1024x1024"
3. May need to iterate 2-3 times for best results

### For Midjourney
1. Use the main prompt
2. Add parameters: `--style raw --v 6 --ar 1:1`
3. For more icon-like: add `--stylize 50`

### For Stable Diffusion
1. Use the detailed prompt (Option 1)
2. Add negative prompt: "realistic, photographic, blurry, low quality, text, watermark"
3. Recommended settings: Steps 30-50, CFG Scale 7-8

## Next Steps
